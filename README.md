# mjanowiak_neo4j_genaiapps
This project contains code for a Neo4J Langchain Chat bot using a Star Wars dataset from Kaggle.

## Data

- Locally I have a `data/` folder which contains the download from the `The Star Wars Dataverse` dataset located here -> https://www.kaggle.com/datasets/jsphyg/star-wars

## Notebooks

- `starwars_neo4j_chat.ipynb` is the main notebook. This notebook loads the data from neo4J
